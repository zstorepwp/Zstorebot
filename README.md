# بوت متجر تليجرام (Zstore)

هذا البوت مصمم لإدارة متجر إلكتروني بسيط على تليجرام مع نظام رصيد وشحن.

## المميزات:
- **نظام رصيد:** يشحن المستخدم رصيده أولاً ثم يشتري.
- **شحن تلقائي:** عبر Cryptomus (يحتاج لإعداد Webhook للتأكيد التلقائي الكامل، حالياً يولد روابط دفع).
- **شحن يدوي:** يمكن للأدمن إضافة رصيد لأي مستخدم عبر ID الخاص به.
- **إدارة المنتجات:** إضافة منتجات (اسم، سعر، محتوى ككود أو رابط).
- **أمان:** لوحة التحكم تظهر للأدمن فقط.

## كيفية التشغيل:
1. تأكد من تثبيت المكتبات المطلوبة:
   ```bash
   pip install python-telegram-bot requests
   ```
2. قم بتشغيل قاعدة البيانات أولاً:
   ```bash
   python database_setup.py
   ```
3. قم بتشغيل البوت:
   ```bash
   python bot.py
   ```

## ملاحظة حول Cryptomus:
لتفعيل الشحن التلقائي بالكامل (تحديث الرصيد فور الدفع)، يجب إعداد `callback_url` في حساب Cryptomus الخاص بك وتوجيهه إلى سيرفر يستقبل التنبيهات ويحدث قاعدة البيانات. النسخة الحالية تقوم بإنشاء روابط الدفع.
